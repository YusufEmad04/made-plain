# Engines of Growth — Plan v5 Review Loop

## Pass 1: First-Time Viewer Critique

The new plan is much clearer than v4 because it defines terms before using shorthand and replaces ad-like logos with neutral examples. It also fixes the biggest v4 issue: metaphors now have explicit business labels.

Remaining risks:

1. **Scene 02 may still become a dense slide.** Three columns plus questions plus inequalities could be too much if all visible at once. It should reveal one engine at a time, then settle into a compact map.
2. **Scene 07 has a lot of definition text.** CAC and LTV definitions are necessary, but must be rendered as large rows with short phrases, not paragraph captions.
3. **Scene 08 could run long visually.** Three failure stories inside one scene are useful, but each must get a full-screen panel. No three-row layout.
4. **Diagnostic scene must avoid too much motion.** It should be a calm checklist, not a fast quiz.
5. **Length target is acceptable only if pacing is calm.** A 2:25-2:45 reel can work for educational content, but only if visuals are clean and not crowded.

## Pass 1 Fixes To Apply

- Change Scene 02 from static three columns to sequential reveals: Sticky first, Viral second, Paid third, then shrink into the top progress strip.
- Add hard typography constraints to the plan: primary statements 76px+, diagram labels 42px+, no text under 34px except decorative ticks.
- In Scene 07, show CAC and LTV as row titles with short one-line definitions only:
  - `CAC = cost to get one customer`
  - `LTV = value from that customer over time`
- In Scene 08, explicitly require panel timing: A, B, C full-screen, then a final three-engine summary. No simultaneous row grid.
- In Scene 09, show one question and one answer tile at a time.

## Pass 2: Visual Safety Critique

The new plan still relies on several diagrams with many parts. The safest version should keep every scene to one of these layouts:

1. One large diagram + one large sentence.
2. Two-number comparison + one consequence line.
3. One question card + one answer.

Anything else risks reverting to v4's crowded explainer-deck feeling.

## Pass 2 Fixes To Apply

- Add a "layout discipline" section to the plan.
- Make the progress strip optional in the first two scenes and compact thereafter.
- Require smoke stills for every reveal phase, not just one midpoint.

## Pass 3: Script Readiness Critique

The plan is ready for scripting if the narration remains calm, literal, and avoids hype. The script should not use phrases like "actually grow," "forever," "turns on its own," or anything that sounds like a growth-hack promise. Use concrete verbs: join, cancel, invite, pay, cost, earn.

## Pass 3 Fixes To Apply

- Keep narration under roughly 430 words.
- Use simple, direct terms repeatedly: new customers, churn, active customers, users, audience, CAC, LTV.
- Never name an acronym without an on-screen definition in the same scene.
